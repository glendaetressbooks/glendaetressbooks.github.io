# Glenda E. Tress Books homepage

GitHub Pages homepage for Glenda E. Tress Books.

## Publish

Upload every file in this folder to the root of the `glendaetressbooks.github.io` repository. All four cover images are deliberately kept at the root so they can be uploaded in one batch.

The Hedgehog card links to the existing book page. The other three cards are marked “Coming soon” and can be linked as their landing pages are published.
